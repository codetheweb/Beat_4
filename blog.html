Part #1 of Alphabet Game

by nxt, May 13, 2014, 6:02 pm

Well, here's the barebones HTML and CSS for my alphabet game. I'll be adding the javascript part of it over the next few days. If anyone has any suggestions, just say so in the comments. I might eventually move this over to Github if it generates enough interest, but for now it's just too much of a hassle.
HTML:
Code:
<!DOCTYPE html>
<html>

<head>
<title>Alphabet Game</title>
<link href="http://www.brainybrainiac.com/alpha/css/main.css" type="text/css" rel="stylesheet">
</head>

<body>
<button id="button">A</button>
</body>
</html>



CSS:
Code:
@font-face
{
font-family: League Gothic;
src: url(http://www.brainybrainiac.com/alpha/fonts/LeagueGothic-Regular.otf);
}
body {
   font-size:100%;
   background-color: #140e0e;
}
#button {
   transition: background-color 0.5s ease;
    width:200px;
    height:200px;
    position:absolute;
    left:50%;
    top:50%;
   margin-left:-100px;
   margin-top:-100px;
   border-radius:12px;
   background-color:#35A3BE;
   color:#140e0e;
   border:none;
   font-family:League Gothic;
   font-size: 1000%;
   text-align:center;
}
#button:active, #button:hover {
   background-color:#174071;
   color:#140e0e;
   border:none;
}


View the live demo at http://brainybrainiac.com/alpha/ and download the lastest files at http://brainybrainiac.com/alpha/alpha.zip
